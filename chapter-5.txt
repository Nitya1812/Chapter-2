6.
What will this code print? Explain why.

function test() {
  return;
  console.log("Done");
}
console.log(test());

ans-- undefined
reason -- When JavaScript sees return;, it immediately exits the function , console.log("Done") never runs.
-- Since no value is returned, the function returns undefined.

9.
What is the difference in how this works in a regular function vs. an arrow function?

ans -- 
--> Regular Function
- this depends on how the function is called.
- It refers to the object that calls the function.

--> Arrow Function
- this is taken from the surrounding (parent) scope.
- It does NOT have its own this.

10.
Write a function factorial(n) that returns the factorial of n (e.g., factorial(5) returns 120).
ans -- 
function factorial(n) {
  let result = 1;
  for (let i = 1; i <= n; i++) {
    result *= i;
  }
  return result;
}
